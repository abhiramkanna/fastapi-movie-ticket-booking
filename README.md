# 🎬 Movie Ticket Booking API (FastAPI)

## 📌 Project Description
This project is a Movie Ticket Booking System built using FastAPI.  
It allows users to:
- View movies
- Add and manage movies
- Filter, search, and sort movies
- Create shows
- Book tickets
- View bookings

---

## 🚀 Features Implemented

### ✅ Day 1: GET APIs
- Home route
- Get all movies
- Get movie by ID
- Summary endpoint

### ✅ Day 2: POST + Validation
- Add movie using Pydantic
- Input validation using Field()

### ✅ Day 3: Helpers + Filter
- Helper functions (find_movie, calculate_discount)
- Filter movies using query parameters

### ✅ Day 4: CRUD Operations
- POST (create)
- PUT (update)
- DELETE (with business rule)

### ✅ Day 5: Workflow
- Create show
- Book tickets
- View bookings

### ✅ Day 6: Advanced Features
- Search movies (case-insensitive)
- Sort movies (price/rating)
- Pagination
- Combined browse endpoint
